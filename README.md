# Adriano Lenaz — CV Website

Personal CV and portfolio site for Adriano Lenaz Zenteno (ADRIP Disruptive Media).

## Pages

- `index.html` — CV: specialization, experience, research, timeline
- `portafolio.html` — Portfolio: graphic journalism, branding, editorial, installation, posters

## Run locally

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Stack

Plain HTML + CSS + vanilla JS. No build step, no dependencies.
Images hosted on Cloudinary.
